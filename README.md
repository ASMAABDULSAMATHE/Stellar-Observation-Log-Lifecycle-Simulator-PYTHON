# STELLAR OBSERVATION LOG & LIFECYCLE SIMULATOR
<p>This console based program is designed to log in the observed stars with dates and times .</p>
<p>The lifecycle simulator advances the star to its next stage in evolution.</p>

## Features
- Add  a star to the log
- View all stars
- Search for a star
- Simulate the lifecycle of a star
- Delete the star data

## Installation
- Clone the repository:
<p>https://github.com/ASMAABDULSAMATHE/Stellar-Observation-Log-Lifecycle-Simulator-PYTHON.git</p>

- Change Folder : cd Stellar-Observation-Log-Lifecycle-Simulator-PYTHON
- Run code: python main.py
